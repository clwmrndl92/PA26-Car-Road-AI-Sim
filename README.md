# PA26 Car Road AI Simulator

펄어비스 여름 인턴십 차량 교통 흐름 AI 시뮬레이터


## 개요

실제 도로 같은 환경에서의 자연스러운 차량 운행과 교통 흐름을 시뮬레이션하는 프로젝트입니다.  
단순한 경로 추종을 넘어, 주변 교통 상황에 반응하는 자율적인 차량 행동을 구현합니다.   

## 주요 기능

- **강체 물리 시뮬레이션** — 이동 및 충돌 시 간단한 물리 처리
- **차선 추종 / 변경 / 합류 / 추월** — 기본적인 도로 주행 행동, 차선간 혹은 도로 밖에서 합류
- **차선 스플라인 기반 길찾기** — 차선 스플라인을 그래프로 구성해 경로 탐색, 교차로 포함
- **교통 흐름 반응** — 앞 차량 상황에 따른 제동 및 출발, 신호 준수
- **돌발 상황 대처** — 예상치 못한 상황에 대한 자연스러운 회피 및 반응
- **주차 기능** — 지정된 주차칸을 인식해 자동으로 진입 및 정차
- **운전 스타일 시뮬레이션** — 차량별 개성 부여 (난폭 운전, 안전 운전, 느린 운전 등 파라미터 기반 다양화)
- **다수 차량 최적화** *(추가 목표)* — 공간 분할, 거리 기반 AI 디테일 차등화, 렌더링 LOD, ECS 적용
