# PA26 Car Road AI Simulator

펄어비스 여름 인턴십 차량 교통 흐름 AI 시뮬레이터


## 개요

실제 도로 환경에서의 자연스러운 차량 운행과 교통 흐름을 시뮬레이션하는 프로젝트입니다.  
단순한 경로 추종을 넘어, 주변 교통 상황에 반응하는 자율적인 차량 행동을 구현합니다.

## 주요 기능

- **차선 추종 / 합류 / 추월** — 기본적인 도로 주행 행동
- **교통 흐름 반응** — 앞 차량 상황에 따른 제동 및 출발
- **돌발 상황 대처** — 예상치 못한 상황에 대한 자연스러운 회피 및 반응
- **복셀 기반 길찾기** — 3D 공간을 복셀로 분할한 경로 탐색
- **강체 물리 시뮬레이션** — 이동 및 충돌 시 간단한 물리 처리
- **운전 스타일 시뮬레이션** — 차량별 개성 부여 (난폭 운전, 안전 운전, 느린 운전 등 파라미터 기반 다양화)
- **다수 차량 최적화** *(추가 목표)* — 거리 기반 AI 디테일 차등화 (원거리: 등속 단순 이동 / 근거리: 다양한 주행 스타일 및 상호작용)

### 8주 개발 계획

| 주차 | 목표 |
|------|------|
| **1주차** | DirectX 11 렌더링 기반 구축, 차량/도로 모델 출력 |
| **2주차** | 차량 이동, 조향, 가속/제동 (기본 강체 물리) |
| **3주차** | 복셀 맵 구성 + 길찾기 |
| **4주차** | 차선 추종 AI + 운전 스타일 설계 및 적용 |
| **5주차** | 교통 흐름 AI — 앞차 감지, 안전거리 유지, 제동 |
| **6주차** | 추월 / 합류 / 돌발 대처 |
| **7주차** | 다수 차량 통합, 전체 시뮬레이션 테스트 |
| **8주차** | 안정화 및 폴리싱, LOD 최적화 *(시간 여유 시)* |
